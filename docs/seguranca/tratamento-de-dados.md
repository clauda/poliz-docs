# Tratamento de Dados

Esta página explica como o Poliz processa os dados pessoais coletados, seguindo as exigências da Lei Geral de Proteção de Dados (LGPD).

## Bases legais para processamento

Todo processamento de dados pessoais precisa ter uma base legal. No Poliz, utilizamos as seguintes:

### Consentimento
Para dados demográficos opcionais que você pode fornecer ao verificar sua conta. Você escolhe fornecer ou não, e pode retirar o consentimento a qualquer momento.

### Execução de contrato
Para fornecer os serviços básicos da plataforma: autenticação, personalização do feed, participação em pesquisas.

### Legítimo interesse
Para melhorias de segurança, prevenção de fraudes e análises internas que não afetam seus direitos fundamentais.

### Cumprimento de obrigação legal
Quando a lei exige que mantenhamos determinados registros.

## Como usamos seus dados

### Autenticação
Seus dados de login são usados exclusivamente para permitir acesso à sua conta.

### Personalização
Seu histórico de navegação e preferências são usados para mostrar conteúdo mais relevante para você.

### Pesquisas
Suas respostas são anonimizadas e agregadas para formar estatísticas. Ninguém sabe como você especificamente respondeu.

### Comunicação
Enviamos notificações sobre a plataforma e atualizações importantes. Comunicações de marketing só são enviadas com seu consentimento explícito.

## O que não fazemos

Reforçamos algumas práticas que o Poliz não adota:

- Não vendemos dados pessoais para terceiros
- Não compartilhamos dados individuais com patrocinadores
- Não usamos algoritmos para manipular sua opinião
- Não rastreamos sua atividade fora da plataforma

## Com quem compartilhamos

### Processadores autorizados
Serviços essenciais para funcionamento da plataforma: hospedagem de dados, autenticação, envio de emails. Esses parceiros são contratualmente obrigados a proteger seus dados.

### Autoridades
Apenas quando legalmente exigido, como em investigações criminais com ordem judicial.

### Parceiros de pesquisa
Apenas dados agregados e anônimos. Nunca informações que possam identificar você.

## Retenção de dados

### Conta ativa
Mantemos seus dados enquanto você usar a plataforma.

### Conta inativa
Após 2 meses sem acesso, seus dados são anonimizados.

### Conta excluída
Seus dados são removidos em até 30 dias após solicitação.

### Respostas de pesquisas
São anonimizadas imediatamente após registro. Os agregados são mantidos indefinidamente para fins históricos.

### Logs de segurança
Mantidos por 6 meses para detectar abusos, depois são convertidos em estatísticas anônimas.

## Seus dados são seus

Você sempre pode acessar, corrigir, exportar ou excluir seus dados. Não criamos barreiras artificiais para dificultar isso. Se precisar de ajuda, nossa equipe está disponível para orientar.
