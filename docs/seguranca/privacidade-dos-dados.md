# Privacidade dos Dados

A privacidade é um direito fundamental. No Poliz, não tratamos isso como formalidade legal, mas como valor central do projeto.

## Nosso compromisso

Nunca vendemos seus dados pessoais. Nunca venderemos. Esse é um compromisso inegociável, independente do modelo de negócio que adotarmos no futuro.

Acreditamos que é possível construir uma plataforma sustentável respeitando integralmente a privacidade dos usuários. E estamos dispostos a provar isso.

## O que coletamos

Coletamos apenas o mínimo necessário para o funcionamento da plataforma.

### Visitantes (sem conta)
- Dados de navegação anônimos
- Preferências armazenadas localmente no seu dispositivo
- Endereço IP anonimizado para estatísticas

### Conta Básica
- Nome fornecido pela rede social de login
- Email (nunca exibido publicamente)
- Identificador único da rede social
- Data de criação da conta

### Conta Verificada (opcional)
Se você fornecer dados adicionais, sua conta será verificada, o que nos ajuda a entender melhor o cenário político brasileiro. Os dados adicionais são:
- Data de nascimento
- Estado
- Gênero
- Escolaridade
- Faixa de renda
- Ocupação

Esses dados são opcionais e usados exclusivamente para segmentação estatística em pesquisas.

## Seus direitos sob a LGPD

A Lei Geral de Proteção de Dados garante a você diversos direitos. No Poliz, facilitamos o exercício de todos eles:

### Acesso
Você pode solicitar uma cópia de todos os dados que temos sobre você a qualquer momento.

### Correção
Se algum dado estiver errado ou desatualizado, você pode pedir a correção.

### Exclusão
Você pode solicitar a exclusão da sua conta e todos os dados associados. Faremos isso em até 30 dias.

### Portabilidade
Você pode exportar seus dados em formato legível para levar para outra plataforma.

### Revogação
Se você deu consentimento para algo específico, pode revogá-lo a qualquer momento.

### Oposição
Você pode se opor a determinados tipos de processamento dos seus dados.

Para exercer qualquer desses direitos, envie um email para privacidade@poliz.com.br. Respondemos em até 15 dias.

## O que não fazemos

É importante deixar claro o que o Poliz não faz com seus dados:

- Não vendemos nem alugamos dados pessoais
- Não compartilhamos dados individuais com políticos ou partidos
- Não rastreamos você em outros sites
- Não usamos seus dados para influenciar seu voto

## Transparência constante

Se algo mudar na forma como tratamos dados, você será notificado com antecedência. Mudanças significativas na política de privacidade são comunicadas com pelo menos 30 dias de antecedência.
