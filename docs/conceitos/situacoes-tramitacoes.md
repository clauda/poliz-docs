# Situação e Tramitação de Projetos

Este guia explica, em **linguagem clara e direta**, o que significam os termos usados para descrever o andamento de projetos de lei (proposições) na Câmara dos Deputados.

A ideia é simples: **você não precisa entender juridiquês para saber se um projeto está andando, parado ou se já acabou**.

## A diferença básica

* **Situação** → mostra **como o projeto está agora** (estado atual)
* **Tramitação** → mostra **o caminho que o projeto percorreu** (histórico de etapas)

Pense assim:

* Situação = *foto do momento*
* Tramitação = *filme completo*

## Situação — “Como está o projeto agora?”

| Situação oficial                    | O que isso quer dizer, em palavras simples                  |
| ----------------------------------- | ----------------------------------------------------------- |
| **Em tramitação**                   | O projeto ainda está sendo analisado                        |
| **Aguardando despacho**             | Parado, esperando alguém decidir o próximo passo            |
| **Aguardando parecer**              | Parado, esperando uma opinião técnica                       |
| **Aguardando votação**              | Na fila para ser votado                                     |
| **Aguardando apreciação**           | Parado, aguardando análise                                  |
| **Pronta para pauta**               | Pode ser votado, mas ninguém colocou na agenda              |
| **Arquivada**                       | O projeto foi encerrado sem virar lei                       |
| **Arquivada ao fim da legislatura** | O projeto morreu porque o mandato acabou                    |
| **Retirada pelo autor**             | O próprio autor desistiu do projeto                         |
| **Rejeitada**                       | O projeto foi votado e perdeu                               |
| **Prejudicada**                     | Perdeu o sentido ou foi substituído por outro               |
| **Transformada em norma jurídica**  | Virou lei                                                   |
| **Aguardando sanção**               | Aprovado pelo Congresso, esperando assinatura do presidente |
| **Vetada**                          | O presidente barrou o projeto                               |
| **Promulgada**                      | Virou lei sem precisar da assinatura do presidente          |

### Dica rápida

Toda situação que começa com **"Aguardando…"** significa, na prática, que o projeto está **parado**.

## Tramitação — “O que já aconteceu com o projeto?”

A tramitação mostra as etapas pelas quais o projeto passou desde que foi criado.

| Tramitação oficial                   | Tradução para linguagem comum          |
| ------------------------------------ | -------------------------------------- |
| **Apresentada**                      | O projeto foi criado                   |
| **Despachada à comissão**            | Enviado para análise técnica           |
| **Distribuída às comissões**         | Enviado para vários grupos analisarem  |
| **Aguardando designação de relator** | Parado, falta alguém assumir a análise |
| **Relator designado**                | Alguém ficou responsável por analisar  |
| **Aguardando parecer do relator**    | Parado, esperando relatório            |
| **Parecer apresentado**              | A análise técnica ficou pronta         |
| **Aprovada na comissão**             | Passou nessa etapa                     |
| **Rejeitada na comissão**            | Foi barrada nessa fase                 |
| **Pronta para pauta**                | Pode ser votada, se for chamada        |
| **Incluída na pauta**                | Vai ser votada                         |
| **Retirada de pauta**                | A votação foi adiada                   |
| **Aprovada em plenário**             | Passou na votação principal            |
| **Rejeitada em plenário**            | Foi votada e perdeu                    |
| **Encaminhada ao Senado**            | Seguiu para a próxima fase             |
| **Recebida no Senado**               | O Senado começou a analisar            |
| **Devolvida à Câmara**               | Voltou com mudanças                    |
| **Enviada à sanção**                 | Aprovada pelo Congresso                |
| **Sancionada**                       | Virou lei                              |
| **Vetada (total ou parcialmente)**   | O presidente barrou tudo ou parte      |
| **Arquivada**                        | Processo encerrado                     |

## Classificação resumida (para leitura rápida)

Para facilitar ainda mais, muitos projetos podem ser entendidos usando apenas estas categorias:

* **Parado** → aguardando alguma ação
* **Em análise** → sendo discutido em comissões
* **Na fila para votar** → pronto, mas ainda não votado
* **Em votação** → sendo decidido
* **Aprovado** → passou
* **Rejeitado** → perdeu
* **Arquivado** → encerrado
* **Virou lei** → concluído com sucesso

## Para não confundir... 

* Situação mostra **o agora**
* Tramitação mostra **o histórico**
* "Aguardando…" quase sempre significa **ninguém mexeu ainda**
* "Arquivada" ou “Rejeitada” significa **acabou**
* "Sancionada" ou “Promulgada” significa **virou lei**

Este guia foi criado para ajudar qualquer pessoa — mesmo sem conhecimento jurídico — a entender rapidamente o que está acontecendo com um projeto de lei.
